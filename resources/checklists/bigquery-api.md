# Checklist de ativação da API do BigQuery

1. Criar projeto no GCP.
2. Ativar faturamento temporário (créditos gratuitos).
3. Habilitar a API do BigQuery via console ou `gcloud`.
4. Conceder permissões mínimas (ex: `BigQuery User`).
5. Confirmar quotas e alertas de custo.

Atualize os passos com os nomes dos recursos usados no seu ambiente.
