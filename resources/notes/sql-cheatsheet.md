# Cheatsheet de SQL no BigQuery

- `SELECT ... FROM ... WHERE ...` (DQL básica)
- `INSERT INTO` / `UPDATE` / `DELETE` (DML)
- `CREATE TABLE` / `CREATE VIEW` (DDL)
- `GROUP BY` / `ORDER BY` / `LIMIT`
- `JOIN` e `UNION` para relacionar tabelas

Complete com atalhos e padrões que facilitaram seu aprendizado.
