# Módulo 8 - Agrupamento e Junção de Tabelas

## Objetivo
Entender como relacionar múltiplas tabelas e sumarizar dados para obter insights agregados.

## Conteúdo programático
- Agrupamento de dados com `GROUP BY`
- Junção de tabelas com `JOIN` (INNER, LEFT)
- União de tabelas com `UNION` e `UNION ALL`
- Junção de tabelas com `JOIN` (RIGHT, FULL)
- Subconsultas básicas

## Atividades recomendadas
1. Construir consultas que combinem joins e agregações para responder perguntas de negócio.
2. Testar `UNION` vs `UNION ALL` e descrever quando usar cada um.
