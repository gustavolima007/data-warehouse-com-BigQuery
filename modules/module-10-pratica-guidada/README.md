# Módulo 10 - Prática Guiada - Análise de Dados de Vendas

## Objetivo
Aplicar os conceitos aprendidos em uma análise prática envolvendo dados de vendas fictícios.

## Conteúdo programático
- Carregamento de dados de vendas fictícios
- Consultas para análise por produto, região e período
- Criação de painel com indicadores de vendas no Looker Studio

## Atividades recomendadas
1. Carregar o dataset de vendas e construir consultas para comparar regiões e períodos.
2. Estruturar métricas-chave (ex: receita, ticket médio, unidades vendidas) antes de criar o painel.

## Recursos
- Consultas para análise macro: `resources/sql/analises_modulo_10.sql`.
- Dados de apoio: `resources/data/vehicle_price_prediction_vehicle_price_prediction.csv`.
