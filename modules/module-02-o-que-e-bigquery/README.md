# Módulo 2 - O que é BigQuery?

## Objetivo
Apresentar o BigQuery, seus diferenciais e como ele se encaixa em projetos de dados modernos.

## Conteúdo programático
- Visão geral do BigQuery
- Cenários de uso e diferenciais do BigQuery
- BigQuery versus modelos convencionais de banco de dados

## Atividades recomendadas
1. Explorar a interface do BigQuery Studio e observar como ele organiza datasets e tabelas.
2. Comparar performance e custos com sistemas tradicionais de SQL originais.
