# Módulo 3 - Criação de um Projeto no GCP

## Objetivo
Configurar o ambiente no Google Cloud para começar a usar o BigQuery com segurança e custos controlados.

## Conteúdo programático
- Acesso ao Google Cloud Platform
- Ativação de créditos no GCP
- Ativação da API do BigQuery

## Atividades recomendadas
1. Criar um projeto gratuito no GCP e habilitar a faturação mínima para testes.
2. Anotar em `resources/notes/gcp-setup.md` os passos, limites e alertas configurados.

## Recursos
- Console do GCP: https://console.cloud.google.com
- Checklist para ativação da API: `resources/checklists/bigquery-api.md`.
