# Módulo 9 - Visualizações e Painéis

## Objetivo
Criar visualizações e painéis simples para comunicar análises de dados com clareza.

## Conteúdo programático
- Visualizações para simplificação de consultas
- Exploração de dados com o BigQuery Studio
- Criação de painéis básicos no Looker Studio

## Atividades recomendadas
1. Criar visualizações no BigQuery Studio para entender agregações e tendências.
2. Publicar um painel simples no Looker Studio conectando-se a uma tabela do BigQuery.

## Recursos
- Consulta de exemplo: `resources/sql/table_olist_looker.sql`.
- Dataset fictício em `resources/data/boardgame-geek-dataset_organized.csv`.
