# Módulo 6 - Introdução à Linguagem SQL

## Objetivo
Dominar os fundamentos de SQL usados no BigQuery, contemplando os principais comandos DQL, DML e DDL.

## Conteúdo programático
- Sintaxe básica do SQL no BigQuery
- SQL DQL (Data Query Language)
- SQL DML (Data Manipulation Language)
- SQL DDL (Data Definition Language)
- Operadores relacionais
- Visão geral das consultas SQL

## Atividades recomendadas
1. Criar consultas básicas usando os operadores relacionais e experimentar `ORDER BY`, `LIMIT` e filtros.
2. Documentar padrões reutilizáveis para DQL, DML e DDL no repositório `resources/notes/sql-cheatsheet.md`.

## Recursos
- Exemplos SQL: `resources/sql/dql_roxschool_cars.sql`, `resources/sql/dml_roxschool_cars.sql`, `resources/sql/ddl_roxschool_cars.sql`.
