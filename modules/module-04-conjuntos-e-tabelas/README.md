# Módulo 4 - Criação de conjuntos de Dados e Tabelas

## Objetivo
Aprender a estruturar conjuntos de dados e modelos de tabelas para armazenar informações no BigQuery.

## Conteúdo programático
- Criação de conjuntos de dados
- Tipos de tabelas (nativas, externas, visualizações)
- Configuração de estruturas e tipos de dados

## Atividades recomendadas
1. Criar datasets de teste com tabelas nativas e externas para comparar comportamentos.
2. Registrar os tipos de dados usados e seus limites em `resources/notes/schema-checklist.md`.

## Recursos
- `resources/sql/ddl_roxschool_olist_database.sql` como exemplo de criação de esquema.
- Documentação de tipos do BigQuery.
