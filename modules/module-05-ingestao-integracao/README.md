# Módulo 5 - Ingestão e Integração de Dados no BigQuery

## Objetivo
Compreender como levar dados para o BigQuery usando diferentes fontes e ferramentas de integração.

## Conteúdo programático
- Upload de arquivos CSV e JSON no BigQuery
- Ingestão através do Google Cloud Storage
- Transferência de dados com Data Transfer Service
- Integração de dados com processos de ETL e automações

## Atividades recomendadas
1. Executar cargas manuais de CSVs e JSONs (por exemplo, `vehicle_price_prediction_vehicle_price_prediction.csv`) e analisar os logs.
2. Configurar um fluxo de transferência automático para um bucket com dados de exemplo.

## Recursos
- Dados de exemplo em `resources/data/`.
- Notas sobre ingestão: `resources/notes/ingestao.md`.
