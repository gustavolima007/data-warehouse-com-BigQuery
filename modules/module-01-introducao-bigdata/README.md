# Módulo 1 — Introdução ao Big Data

## Objetivo

Compreender os conceitos centrais de Big Data, Data Warehouse, Data Lake e arquiteturas híbridas, identificando quando utilizar cada abordagem dentro de um pipeline analítico.

## Conteúdo programático

- Panorama Big Data e os 5 V's.
- Diferenças entre Data Warehouse, Datamart, Data Lake e Data Lakehouse.
- Arquiteturas de referência com BigQuery.
- Casos de uso por indústria.

1. **Mapa mental**: crie um quadro relacionando fontes de dados, zona de staging, camadas consumidas e ferramentas de visualização.
2. **Glossário**: documente os principais termos (Data Warehouse, Datamart, Data Lake etc.) com exemplos reais.
3. **Casos de uso**: mapeie, em até três cenários da sua empresa, qual arquitetura resolve cada problema.
