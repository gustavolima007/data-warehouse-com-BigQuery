# Módulo 7 - Filtragem e Ordenação de Dados

## Objetivo
Aplicar filtros precisos e controlar a ordem do retorno das consultas para ganhar clareza nos resultados.

## Conteúdo programático
- Operadores lógicos e de comparação
- Ordenando dados com `ORDER BY`
- Controle de resultados com `LIMIT`

## Atividades recomendadas
1. Usar os operadores lógicos e de comparação em conjunto para montar filtros complexos.
2. Criar consultas com `ORDER BY` e `LIMIT` e explicar o impacto na performance.
